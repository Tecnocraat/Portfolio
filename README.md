# IT Professional Portfolio

This repository contains a collection of software projects developed primarily in **C and C++**, organized by version and technical evolution.

## Projects

### Currency Converter

A currency conversion application that evolves through multiple versions, each introducing new technical concepts and improvements.

#### Roadmap

- **v1.0** — Console-based conversion using fixed exchange rates.
- **v2.0** — Real-time exchange rates via external API
- **v2.1** — Data persistence using PostgreSQL
- **v3.0** — Graphical user interface built with Qt (in development)


> This portfolio is under continuous development and expansion.
