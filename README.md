# Portfolio de Projetos em C/C++

Este repositório reúne projetos desenvolvidos em C e C++, organizados por versões e evolução técnica.

## Projetos
### Conversor de Moedas
Aplicação para conversão de moedas, evoluindo de uma versão console simples até uma aplicação gráfica com Qt.

### Roadmap
- v1.0: Conversão com taxas fixas (console)
- v2.0: Taxas em tempo real via API
- v2.1: Persistência em PostgreSQL
- v3.0: Interface gráfica com Qt (em desenvolvimento)

> Projeto em constante evolução.
